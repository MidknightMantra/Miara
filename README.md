## âš™ï¸ Configuration

Edit the `config.json` file to customize your bot:
```json
{
  "prefix": ".",
  "owner": "your-number@s.whatsapp.net",
  "botName": "Miara",
  "autoRead": true,
  "autoReconnect": true
}
```

### Configuration Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `prefix` | string | Command prefix for the bot | `.` |
| `owner` | string | Bot owner's WhatsApp number | - |
| `botName` | string | Display name for the bot | `Miara` |
| `autoRead` | boolean | Automatically read messages | `true` |
| `autoReconnect` | boolean | Auto-reconnect on disconnect | `true` |
