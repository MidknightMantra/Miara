## ğŸš€ Installation

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- A WhatsApp account

### Steps

1. Clone the repository
```bash
git clone https://github.com/MidknightMantra/Miara.git
cd Miara
```

2. Install dependencies
```bash
npm install
```

3. Configure the bot
```bash
cp config.example.json config.json
# Edit config.json with your settings
```

4. Start the bot
```bash
npm start
```

5. Scan the QR code with your WhatsApp to connect

## ğŸ¯ Usage

Once the bot is running and connected:

1. Send a message to your bot's WhatsApp number
2. Use the `.help` command to see available commands
3. Interact with the bot using the command prefix (default: `.`)

## âš™ï¸ Configuration

Edit the `config.json` file to customize your bot:
```json
{
  "prefix": ".",
  "owner": "your-number@s.whatsapp.net",
  "botName": "Miara",
  "autoRead": true,
  "autoReconnect": true
}
```

### Configuration Options

| Option | Type | Description |
|--------|------|-------------|
| `prefix` | string | Command prefix for the bot |
| `owner` | string | Bot owner's WhatsApp number |
| `botName` | string | Display name for the bot |
| `autoRead` | boolean | Automatically read messages |
| `autoReconnect` | boolean | Auto-reconnect on disconnect |
